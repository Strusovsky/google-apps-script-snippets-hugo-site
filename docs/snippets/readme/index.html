    deleteRowsByConditional_(sheet, condition)

    Removes rows from a sheet according to the condition

    Kind: global function

    Param Type Description
    sheet Sheet Represents the Sheet that is changing.
    condition conditionCallback The callback that should return true/false state for each row.

    Example

    // Removes all rows where B column contains 10
deleteRowsByConditional_(SpreadsheetApp.getActiveSheet(),
  function(currentValue){
    return currentValue[1] === 10;
  }
);

    conditionCallback : function

    Returns true/false state for each row.

    Kind: global typedef

    Param Type Description
    currentValue Array The current row of the DataRange
    index Number The index of the current row. The DataRange is reversed!
    array Array The DataRange. The DataRange is reversed!
